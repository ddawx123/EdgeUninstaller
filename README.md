## Edge All In One Remover

Test on Windows 11 Insider Preview 29576 and Edge Kernel 150 Stable Version

Usage: execute RunMe.bat

For international user recommendation: Use windows terminal (administrator mode) to avoid Chinese traditional font display issue.
(Right click the start button, and select "Terminal (Admin)".)

Design by myself, but script written by artificial intelligence.

Other information: Original installer for Edge 109 has been provided in release page, it's compatible with NT6.1-6.3, the best version for nt10! Also you can deploy the Edge Legacy (EdgeHTML kernel, the Spartan project) for Windows 11 21H2-23H2, but 24H1-26H2 is not compatible (for my test result).
